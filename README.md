# habit tips
if you want start or keep a habit, follow below may make it easyer(if you want quit bad habit, do the different side of tips)

## what is habit
### we can devide habit to 4 parts:
提示 -> 渴望 -> 回應 -> 獎賞
### 提示
提示 == 提示促使大腦開啟一個行為，如果古人"發現"這樣那樣很有可能附近有免費的好喝的水、食物。也就是說，提示就像C++=的 forward_list ，"刷刷"的聲音代表附近可能有水，"哞哞哞"的聲音代表
附近有免費的牛肉。 forward_list 最終會指向對演化有優勢的事物，當然也可能指向不確定的事物。




"刷刷"聲 --水的提示> 水 --你渴望水(你想活下去)> 活下去
   |
   |
   ---你渴望活下去--你去找水了>找到水了--喝水>真爽
